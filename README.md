# 🗳️ Voter Registration Management System Landing Page

A modern, high-performance, fully responsive landing page created for the **Voter Registration Management System (VRMS)**. It features a clean government-style look and is optimized for easy static deployment.

🔗 Live Demo: [VRMS_LandingPage](https://sagarsingh-git-hub.github.io/Voter_Registration_Management_System-Landing_Page/)

## 🚀 Features

- **Responsive Design**: Works well on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional interface with smooth animations
- **Dark Mode Support**: Built-in dark mode functionality for enhanced user experience
- **Accessibility**: Semantic HTML5 and ARIA-friendly structure
- **Performance Optimized**: Minimal dependencies for fast loading
- **Smooth Animations** – Subtle transitions and hover effects

## 🛠️ Technologies Used

- **HTML5**: Semantic structure & accessibility
- **Tailwind CSS**: Utility-first styling framework
- **Google Fonts**: Space Grotesk & Inter for typography
- **JavaScript**: Interactivity & smooth UI behavior

## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/SagarSingh-git-hub/Voter_Registration_Management_System-Landing_Page.git
   ```

2. Navigate to the project directory:
   ```bash
   cd "Voter Registration Management System Landing Page"
   ```

3. Open `index.html` in your web browser or serve with a local web server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using Live Server (VS Code extension)
   # Right-click index.html and select "Open with Live Server"
   ```
